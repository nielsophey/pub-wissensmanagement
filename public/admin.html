Wiki veröffentlichen

Nach der Wiki-Generierung kann das Ergebnis direkt auf der konfigurierten Plattform veröffentlicht werden:

  • GitHub: Markdown-Dateien werden per API ins Repository gepusht
  • Azure DevOps: Seiten werden im Projekt-Wiki angelegt/aktualisiert

Aufgaben aus TODOs

In den Wiki-Artikeln erkannte TODOs werden automatisch als Aufgaben erstellt:

  • - [ ] Aufgabentext → Checkbox-Syntax
  • TODO: Aufgabentext → Inline-Marker

GitHub: Issues mit konfigurierten Labels
Azure DevOps: Work Items (Task, Bug, etc.)
